Procedure

Description of Research Process Steps


Step

Description

Image

1

Train CNN with Training Dataset Images:

Minimizing a loss function by optimizing parameters in the CNN.
  • The value of the loss function is decreased as the iterations increased.
  • During each iteration, the self-supervised learning goes through the entire
    training data.
  • This allows the CNN to "learn" what a real galaxy merger looks like and identify when the image does not show a merger.

2

Randomly Augument Test Dataset Images:

Augumentation ensures the CNN is trained to identify and read images that are
not completly clear of "distractions."
These distractions include:
  • Rotation
  • Random noise
  • Inverting
Not all images of distant galaxies are of best quality

3

Enter Test Dataset Images Into CNN Input Layer:

  • Apply convolutional layers to input images
  • Use activation functions to introduce non-linearity
  • Reduce dimensionality with pooling layers
  • Produce final output with fully connected layers
  • Adjust weights to minimize output differences
Input -> Convolution -> ReLU -> Convolution -> ReLU -> Pooling ->
Fully Connected -> Output

4

Extract Feature Respresentations and Compute Similarity:

CNN outputs low-dimensional space feature vectors that are respresentations of high-dimensional space images
  • It's hard to compare and read images, but we can do it for feature vectors
  • This allows us to see how similar a picture is to a known merger
Galaxy images can be identified by calculating the cosine similarity score of their feature vector and the vector of a known merger.

5

Conduct Similarity Search:

A similarity search is conducted with a query image based on the similarity scores.
  • This query image is a known galaxy merger
  • The more similar a test image is to a known galaxy merger, the more likely it is to be a galaxy merger
